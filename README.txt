Приложение бот-полиглот

Бот для телеграмма. При подключении бота появляется выбор языка, на который будут переводиться сообщения.
Далее при отправке текста в этот телеграмм-бот, бот будет подключаться к сервисам яндекс-переводчика и
переводить сообщения на выбранный язык.

Languages - перечисление, содержит языки перевода
YandexTranslate - сервис для перевода текста из одного языка в другой. Используется API Yandex Cloud.
TranslateBot - сам телеграм-бот. Используется библиотека telegrambots.
